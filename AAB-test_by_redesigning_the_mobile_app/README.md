# A/A/B-эксперимент для мобильного приложения.

### Задача:

Используя данные с результатами проведенного A/A/B-теста по редизайну мобильного приложения, необходимо выяснить, есть ли смысл менять шрифты.

### Цель исследования:

- изучить поведение пользователей мобильного приложения в воронке продаж
- определить, сколько пользователей доходит до покупки, а сколько — «застревает» на предыдущих шагах
- исследовать результаты A/A/B-эксперимента и принять финальное решение по результатам эксперимента о том, какой шрифт лучше

### Результат:

- **исследовательский анализ:**

Всего 243 713 событий и 7 551 пользователей в логе. В среднем 20 событий приходится на пользователя с поправкой на медианное значение.

В логе хранятся данные за 14 дней - с 25 июля по 7 августа 2019 года. Проанализировав гистограмму распределения событий по дате, пришли к выводу, что мы не можем быть уверены в том, что у нас одинаково полные данные за весь период. Установили реальный временной период лога - неделю, с 1-го по 7-е августа 2019 года и отбросили старые данные.

Убедились, что потери при фильтрации июльских данных незначительные - исключили 1,16 % событий и 0,23 % пользователей, масштаб бедствия невелик. 

- **изучение воронки событий:**

Зафиксировали, что 7 419 пользователей заходили на главную страницу мобильного приложения, что составляет почти 98,5 % от их общего числа. Почти 61 % пользователей хотя бы раз открывали страницу каталога с продуктами, было бы поезно определить, почему 39 % не доходят до данного этапа. Примерно половина клиентов хотя бы раз открывали корзину и около 47 % клиентов видели страницу с информацией об успешной оплате. Реже всего пользователи открывали туториал, порядка 11 % от общего числа клиентов.

Определении следующий порядок событий в воронке продаж:

1. Открытие главного экрана
2. Открытие каталога товаров
3. Открытие корзины
4. Экран успешной оплаты

Не стали включать туториал при расчете воронки событий, в силу отсутствия необходимости или обязательности.

Изучив доли пользоватлей в воронке событий для каждого этапа, зафиксировали следующие доли от числа пользователей на предыдущем этапе воронки: на вотором шаге осталось 62 % пользователей, на третьем шаге конверсия увеличилась до 81 %, на заключительном шаге - максимальная конверсия в 95 %.

Большую часть пользователей - 38 % теряем на этапе просмотра каталога товаров. Имеет смысл разобраться, насколько корректно открывается каталог на различных устройствах и нет ли технических проблем на данном этапе воронки событий. От просмотра главной страницы до успешной оплаты доходят чуть меньше половины пользователей - 48 %.

- **изучение результатов эксперимента:**

В результате проведенного A/A-эксперимента не смогли отвергнуть нулевые гипотезы. Следовательно, между двумя контрольными группами A0 и А1 нет статистически значимых различий, что подтверждает точность проведенного тестирования.

По результатам проверки первой контрольной - А0 и экспериментальной групп, пришли к выводу о том, что новые шрифты не повлияют на изменение поведения пользователей. Не смогли отвергнуть нулывые гипотезы, нет оснований считать конверсии событий разными.

После z-теста для второй контрольной - А1 и экспериментальной групп, вновь отметили нецелесообразность изменения шрифтов. Р-значение для каждого вида события также значительно превышает критический уровент стат. значимости, поэтому нет оснований считать конверсии событий разными.

В результате сравнения объединенной контрольной - А0 + А1 и экспериментальной групп получили такие же выводы по проверке гипотез - не можем считать конверсии событий объединенной контрольной и экспериментальной групп разными.

Резюмируя результаты всех статистических проверок, приняли решение остановить тест и зафиксировать, что поведение пользователей группы В не изменилось, данный сегмент не показал себя лучше контрольных групп. Для каждого вида событий воронки продаж не смогли отвергнуть нулевую гипотезу о равенстве конверсии контрольных и экспериментальной групп, следовательно, не можем считать конверсии событий разными. В силу отсутствия статистической значимости различий в конверсиях событий между сегментами, изменение шрифтов во всём приложении не повлияет на поведение пользователей и не будет эффективным и целесообразным для стартапа, потому что не принесет бизнесу дополнительной выручки.
