# Анализ взаимодействия пользователей с карточками Яндекс.Дзен.

### Задача:

На основе бэкап-файла из базы данных `PostgreSQL` c логами пользователей в Яндекс.Дзене, предоставленных дата-инженерами за 24 сентября 2019 года с 18:28 по 19:00, **построить дашборд** для менеджеров по анализу контента согласно ТЗ, **главной бизнес-задачей** которого является анализ взаимодействия пользователей с карточками Яндекс.Дзен.

 ### Результат:

**Взаимодействие пользователей с карточками с разбивкой по темам карточек.**

Больше всего пользователи взаимодействуют с карточками по темам:

- Наука
- Отношения
- Интересные факты
- Общество
- Подборки

**Взаимодействие пользователей с карточками с  разбивкой по темам источников.**

Топ-5 тем источников карточек:

- Семейные отношения
- Россия
- Полезные советы
- Путешествия
- Знаменитости

**Соотношение тем карточек и тем источников.**

Наибольшее количество взаимодействий пользователей у следующих пар тем карточек и источников:

- Рассказы - Путешествия (4587)
- Общество - Россия (3471)
- Наука - Кино (3279)
- Россия - Россия (2847)
- Подборки - Полезные советы (2795)
